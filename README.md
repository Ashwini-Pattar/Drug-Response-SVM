# Drug Response Classification using SVM

## Project Overview
This project predicts whether a patient responds positively to a drug using a Support Vector Machine (SVM) model.

Machine learning helps pharmaceutical companies analyze patient data and determine drug effectiveness faster than traditional clinical trials.

## Dataset
The dataset contains medical indicators of patients.

Features:
- Drug Dosage
- Systolic Blood Pressure
- Heart Rate
- Liver Toxicity Index
- Blood Glucose Level

Target Variable:
Drug Response
0 = No Response  
1 = Positive Response

## Steps Performed
1. Data Loading
2. Exploratory Data Analysis (EDA)
3. Data Visualization
4. Categorical Encoding
5. Data Preprocessing
6. Support Vector Machine Model Implementation
7. Hyperparameter Tuning
8. Model Evaluation

## Technologies Used
- Python
- Pandas
- NumPy
- Seaborn
- Matplotlib
- Scikit-learn

## Model
Support Vector Machine (SVM) classifier was used to predict drug response based on patient health indicators.

## Output
The model evaluates performance using:
- Accuracy
- Confusion Matrix
- Classification Report
